# RI2_ObjetosGetSetESort
Projeto da matéria de Programação Orientada a Objetos em JavaScript. Este exercício foca na prática de Funções Construtoras e Encapsulamento. Implementa objetos Cliente, TelefoneCelular e Endereco com getters e setters, métodos para formatação de strings (caixa alta/baixa), e uma função para ordenar um Array de clientes alfabeticamente por nome.
